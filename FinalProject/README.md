# CapyBerry 
A mobile puzzle game app that is themed around a Capybara gathering fruits through matching the appropriate orbs. The gameplay is inspired by Puzzles & Dragons, which is a puzzle click and drag matching RPG game. The aim is to get as many orb matches in as possible and achieve a high score before the timer runs out. 

##Quick Start
Upon launch, the user can begin playing by selecting the Start Game button. They will then redirected to the Gameplay screen
